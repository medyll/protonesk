# S4-04 — SMTP routing par expéditeur

**Sprint:** 4 | **Effort:** M | **Status:** planned | **Dépend de:** S4-02

## User Story

En tant qu'utilisateur, je veux envoyer depuis le bon compte Proton selon l'adresse `From:` du mail sortant.

## Acceptance Criteria

- [ ] `ProtonSMTPHandler` extrait `From:` header du message entrant
- [ ] Mappe l'adresse From au label du compte (via `SessionManager`)
- [ ] Si adresse inconnue → utilise compte par défaut (premier compte) + warning log
- [ ] Si compte trouvé → appelle `ProtonSend` de ce compte spécifique
- [ ] Erreur 550 si compte trouvé mais session invalide

## Routing logic

```python
# From: perso@proton.me → label "perso" → session perso → ProtonSend perso
# From: pro@proton.me   → label "pro"   → session pro   → ProtonSend pro
# From: unknown@...     → compte[0] par défaut + warning
```
