# S2-01 — Serveur SMTP local

**Sprint:** 2 | **Effort:** M | **Dépend de:** S1-04 (ProtonSend)

## User Story

En tant qu'utilisateur, je veux pouvoir envoyer un mail depuis n'importe quel client (Thunderbird, n8n, script Python) vers `localhost:1025`, et que ce mail soit automatiquement chiffré et envoyé via mon compte Proton.

## Acceptance Criteria

- [ ] Serveur écoute sur `127.0.0.1:1025` (configurable)
- [ ] Accepte connexion SMTP sans auth (local only)
- [ ] Parse les champs `From`, `To`, `Subject`, `Body` du message entrant
- [ ] Passe le message à `ProtonSend.send_email()` avec chiffrement PGP
- [ ] Retourne code `250 OK` si envoi réussi, `550` si échec
- [ ] Log chaque envoi avec timestamp et destinataire

## Implémentation cible

- `src/smtp_server.py` — classe `ProtonSMTPServer` via `aiosmtpd`
- Handler custom qui intercepte chaque message et appelle `ProtonSend`

## Tests requis

- Test envoi message simple (mock ProtonSend)
- Test parse headers SMTP → dict
- Test réponse 250 sur succès, 550 sur échec
