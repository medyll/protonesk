# S4-03 — IMAP namespace par compte

**Sprint:** 4 | **Effort:** M | **Status:** planned | **Dépend de:** S4-02

## User Story

En tant qu'utilisateur, je veux voir mes dossiers de chaque compte séparément dans Thunderbird.

## Acceptance Criteria

- [ ] `LIST` retourne dossiers préfixés par label : `perso/INBOX`, `pro/Sent`, etc.
- [ ] `SELECT perso/INBOX` → sélectionne INBOX du compte "perso"
- [ ] `SELECT pro/INBOX` → sélectionne INBOX du compte "pro"
- [ ] Parser de mailbox : `"perso/INBOX"` → `label="perso"`, `mailbox="INBOX"`
- [ ] `MultiAccountBridge` remplace `ProtonIMAPBridge` comme glue layer
- [ ] Rétrocompat : si un seul compte, préfixe optionnel (INBOX fonctionne aussi)

## Mapping IMAP

```
* LIST (\HasNoChildren) "/" "perso/INBOX"
* LIST (\HasNoChildren) "/" "perso/Sent"
* LIST (\HasNoChildren) "/" "pro/INBOX"
* LIST (\HasNoChildren) "/" "pro/Sent"
```
