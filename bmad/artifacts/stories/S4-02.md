# S4-02 — Session manager (pool de sessions Proton)

**Sprint:** 4 | **Effort:** M | **Status:** planned | **Dépend de:** S4-01

## User Story

En tant que système, je veux gérer plusieurs sessions Proton actives sans les mélanger.

## Acceptance Criteria

- [ ] `src/session_manager.py` — classe `SessionManager`
- [ ] `SessionManager.connect_all(accounts)` → dict `{label: (auth, session, api_client)}`
- [ ] `SessionManager.get(label)` → retourne session pour ce compte
- [ ] `SessionManager.reconnect(label)` → reconnecte un compte sans toucher les autres
- [ ] `SessionManager.logout_all()` → logout propre de tous les comptes
- [ ] Chaque compte charge ses credentials via `SecretManager(service=f"proton-bridge-{label}")`

## Interface cible

```python
manager = SessionManager()
manager.connect_all(cfg["accounts"])
session = manager.get("perso")  # → ProtonSession
manager.reconnect("pro")        # si session expirée
manager.logout_all()
```
