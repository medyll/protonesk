# S3-01 — TLS auto-gen + IMAP SSL

**Sprint:** 3 | **Effort:** M

## Acceptance Criteria

- [ ] `src/tls.py` génère cert auto-signé (RSA 2048) si absent dans `~/.proton-bridge/certs/`
- [ ] Cert valide 365 jours, SAN localhost + 127.0.0.1
- [ ] `ProtonIMAPServer` accepte paramètre `ssl_context` optionnel
- [ ] Avec ssl_context → connexion TLS native (port 1993 par convention)
- [ ] Sans ssl_context → connexion plaintext (port 1143, comportement actuel)
- [ ] `main.py --tls` génère/charge cert et démarre IMAP en TLS
