**Goal:** Sécuriser les connexions (TLS), ajouter push IMAP (IDLE), simplifier la config.

## Architecture TLS

```
Cert auto-généré au démarrage → ~/.proton-bridge/certs/
IMAP: asyncio.start_server(ssl=ssl_context)  → port 1993 (TLS) ou STARTTLS sur 1143
SMTP: aiosmtpd Controller(ssl=ssl_context)   → STARTTLS sur 1025
```

## Out of scope

- CA signing (self-signed suffice pour usage local)
- IMAP IDLE avec vrai push webhook Proton (polling 30s suffit)
